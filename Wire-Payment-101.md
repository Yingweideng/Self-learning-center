## 什么是电汇支付？

### 🏦 通俗理解

想象一下你要给朋友寄钱，但不想用现金，而是通过银行"打电话"告诉对方银行："请从我账户扣钱，存到我朋友的账户里"。这个"打电话"的过程就是电汇。

###  正式定义

**电汇支付（Wire Transfer）** 是一种通过安全的银行间网络，将资金从一个银行账户电子转移到另一个银行账户的支付方式。

### 🔑 核心特点

| 特点 | 说明 |
|------|------|
| **速度快** | 国内电汇通常当天到账，国际电汇 1-3 个工作日 |
| **安全性高** | 通过加密的银行间网络传输，不可撤销 |
| **金额大** | 适合大额转账，限额较高 |
| **费用较高** | 比 ACH 转账贵，但比支票快 |

## 电汇支付的类型

### 🗺️ 按地域分类

```
┌─────────────────────────────────────────────────────┐
│                  电汇支付                            │
├─────────────────────────┬───────────────────────────┤
│     国内电汇            │      国际电汇             │
│  (Domestic Wire)        │   (International Wire)    │
│                         │                           │
│  • 同一国家内转账       │   • 跨国转账              │
│  • 使用 Fedwire/CHIPS   │   • 使用 SWIFT 网络        │
│  • 当天到账             │   • 1-3 个工作日到账       │
│  • 费用 $15-50          │   • 费用 $30-100+         │
└─────────────────────────┴───────────────────────────┘
```

### 1️⃣ 国内电汇（Domestic Wire Transfer）

**定义**：在同一国家内的两个银行账户之间进行的资金转移。

**美国主要网络**：

| 网络 | 全称 | 特点 | 适用场景 |
|------|------|------|----------|
| **Fedwire** | Federal Reserve Wire Network | 实时全额结算，由美联储运营 | 紧急大额转账 |
| **CHIPS** | Clearing House Interbank Payments System | 净额结算，由清算所运营 | 批量大额支付 |

#### Fedwire vs CHIPS 对比

```
┌────────────────────────────────────────────────────────────────
│ Fedwire（实时全额结算）                                          │
├────────────────────────────────────────────────────────────────┤
│ 工作原理：每笔交易单独处理，立即结算                              │
│                                                                │
│ 银行 A ──────→ 美联储 ──────→ 银行 B                           │
│   $100 万      立即扣款      立即到账                           │
│                                                                │
│ 优点：实时到账，不可撤销                                         │
│ 缺点：流动性要求高                                               │
└────────────────────────────────────────────────────────────────┘

┌────────────────────────────────────────────────────────────────┐
│ CHIPS（净额结算）                                                │
────────────────────────────────────────────────────────────────┤
│ 工作原理：多笔交易轧差后结算                                     │
│                                                                │
│ 银行 A 欠银行 B $100 万                                         │
│ 银行 B 欠银行 A $80 万                                          │
│ ──────────────────────────                                     │
│ 最终：银行 A 只需支付银行 B $20 万                               │
│                                                                │
│ 优点：节省流动性，降低成本                                       │
│ 缺点：日终结算，非实时                                           │
└────────────────────────────────────────────────────────────────┘
```

### 2️⃣ 国际电汇（International Wire Transfer）

**定义**：跨越国界的资金转移，通常涉及货币兑换。

**核心网络**：**SWIFT**（Society for Worldwide Interbank Financial Telecommunication）

```
┌─────────────────────────────────────────────────────────────────┐
│ SWIFT 网络架构                                                   │
├─────────────────────────────────────────────────────────────────┤
│                                                                 │
│  中国银行        代理银行        美国银行                         │
│  (SWIFT:        (SWIFT:        (SWIFT:                          │
│   BKCHCNBJ)     CHASUS33)      BOFAUS3N)                        │
│      │              │              │                            │
│      │──SWIFT 消息──→│──SWIFT 消息──→│                            │
│      │  (MT103)     │  (MT103)     │                            │
│      │              │              │                            │
│      │←─资金流───────│←─资金流───────│                            │
│      │  (通过代理行)  │  (通过 Fedwire) │                            │
│                                                                 │
│  备注：SWIFT 只传递消息，不转移资金！资金通过代理行账户实际划转       │
└─────────────────────────────────────────────────────────────────┘
```

**国际电汇的关键角色**：

| 角色 | 英文 | 职责 |
|------|------|------|
| **汇款人** | Originator | 发起转账的人/企业 |
| **汇款行** | Originating Bank | 汇款人的银行 |
| **代理行** | Intermediary/Correspondent Bank | 中转银行（如有需要） |
| **收款行** | Beneficiary Bank | 收款人的银行 |
| **收款人** | Beneficiary | 接收资金的人/企业 |

## 国内电汇工作流程

### 📊 Fedwire 工作流程

```
┌──────────────────────────────────────────────────────────────────────┐
│                    Fedwire 国内电汇完整流程                           │
└──────────────────────────────────────────────────────────────────────┘

步骤 1: 发起转账
┌─────────────┐
│   汇款人    │ ─→ 登录网银/前往柜台
│  (您)       │     填写收款人信息
└─────────────┘     输入转账金额
       │            确认手续费
       ↓
─────────────┐
│  汇款行     │ ←─ 验证账户余额
│  (您的银行) │     验证身份信息
└─────────────┘     验证反洗钱筛查
       │
       ↓ 步骤 2: 发送支付指令
┌─────────────┐
│   Fedwire   │ ←─ 实时全额结算系统
│   网络      │     立即扣款汇款行账户
└─────────────┘     立即贷记收款行账户
       │
       ↓ 步骤 3: 资金到账
┌─────────────┐
│  收款行     │ ←─ 收到资金
│ (收款人银行)│     更新收款人账户余额
└─────────────┘     发送入账通知
       │
       ↓ 步骤 4: 通知收款人
┌─────────────┐
│   收款人    │ ←─ 收到到账通知
│  (朋友)     │     账户余额增加
└─────────────┘

⏱️ 总耗时：通常几分钟到几小时（工作时间内）
```
